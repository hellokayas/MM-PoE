## Installation
### Install it from PyPI

```bash
pip install mm_poe
```

### Install it from source

```bash
$ git clone https://github.com/souradipp76/MM-PoE.git
$ cd MM-PoE
$ make install
```

## Usage

```bash
$ python -m mm_poe
#or
$ mm_poe
```
